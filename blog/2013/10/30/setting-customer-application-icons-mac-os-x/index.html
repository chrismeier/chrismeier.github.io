Setting Custom Application Icons Mac OS X

Oct 30th, 2013
  • off-topic
  • apple
  • mac
  • design
Comments

Being a fan of the flat design iOS 7 design I wanted to try and make my computer feel a little more “flat”. My first step was to make my app icons on my Dock flat. Below is the end result and what follows is how you can do it.

1. Find the image that you want to use

Obviously you can use any image you want but the easiest way to find one is to search google for icons and in this case “Safari flat icon”.

2. Get the right image

You want to make sure the image is actually an icon and not an image with a white background. When you click the image result you are able to tell if it the image has a transparent background by the fact that it has a checkered backdrop (see below). Once you have found the image that you want click view image (as highlighted below) and then right click and save the image you see in the new page.

3. Apply the icon

Go to the application that you want to apply the icon to. You can view your applications by opening up finder and clicking the Applications folder in the side menu. Once you have found the Application that you want to apply the icon to right-click it and select “Get Info”. Next, click the icon in the Get Info pane (see below). You should now see it highlighted in blue. Now, press Command+V to paste your icon over the old one. Note: You will need to restart your computer OR restart Finder and Dock (Terminal commands “killall Dock” and “killall Finder”) in order for the icons to go into effect.
